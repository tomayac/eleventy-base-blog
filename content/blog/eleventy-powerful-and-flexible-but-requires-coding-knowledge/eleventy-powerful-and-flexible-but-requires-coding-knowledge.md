So, you're looking at Eleventy for your site, huh? It's a pretty sweet tool, seriously fast and you can tweak it to do pretty much anything you want. But, and this is a big "but," if you're more of a writer than a coder, you might find it a bit… much.

See, unlike some other website builders, Eleventy doesn't have that handy little editor you can just hop into and start typing away. Nope. Usually, if you want to write or change anything, you're diving into actual code files using your development software. Now, if you're a developer, that's probably right up your alley. But for folks who just want to get their ideas out there without wrestling with code, it can feel like a real hurdle. It definitely makes things a bit trickier to get started if you're not super chummy with programming languages.
